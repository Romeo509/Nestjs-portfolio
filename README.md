# NestJS Ops Center

## Overview
The **NestJS Ops Center** is a structured initiative aimed at designing, developing, and deploying robust NestJS backend applications. Each project in this repository follows industry best practices, enforces clean architecture principles, and provides real-world backend solutions to enhance skills and expertise in NestJS development.

Each module is well-documented, ensuring clarity in implementation, deployment, and usage. Project-specific **README** files provide technical breakdowns and operational insights.

---

## Objectives

### 1. User Authentication API
- Implement JWT-based authentication using NestJS Passport.
- Support login, registration, password reset, and role-based access control (RBAC).

### 2. Task Management API
- Enable users to create, update, delete, and track tasks.
- Implement filtering by due dates and priority levels.

### 3. Blogging API
- Provide CRUD operations for posts, comments, and user profiles.
- Implement authentication and role-based permissions (admin, user).

### 4. URL Shortener API
- Allow users to shorten URLs and track analytics on click counts.
- Utilize PostgreSQL or MongoDB for data storage.

### 5. File Upload & Management API
- Enable file uploads, deletions, and management using Multer with NestJS.
- Support local or cloud storage (AWS S3, Cloudinary, etc.).

### 6. WebSocket Chat API
- Implement real-time messaging using WebSockets.
- Store and retrieve messages from MongoDB or PostgreSQL.

### 7. E-commerce Backend (Minimal Version)
- Provide APIs for products, orders, users, and payments (Stripe/PayPal integration).
- Implement cart and checkout functionality.

### 8. Bookstore RESTful API
- Offer CRUD operations for books, authors, and categories.
- Implement pagination and search features for efficient data retrieval.

### 9. Rate Limiting API
- Implement middleware for API rate limiting.
- Store request logs in Redis for enhanced security and performance.

### 10. Cryptocurrency Price Tracker API
- Fetch real-time cryptocurrency prices from external APIs (CoinGecko, Binance API).
- Cache responses using Redis for optimized performance.

## Execution Plan
